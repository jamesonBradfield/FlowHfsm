Project Guidelines: Godot 4.x Development
Global Restrictions

    NO.TSCN WRITES: The file extensions .tscn, .scn, .tres, and .res are READ-ONLY.

        Reason: Prevention of UID corruption and dependency breakage.

    Static Typing: All variables and functions must be statically typed.

Context Awareness

    This is a Godot 4.3 project.

    Ignore any training data related to Godot 3.5 or earlier.

    When searching for documentation, prioritize the godot-expert agent.
